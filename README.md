## Iniciando

### Installation

Clone the repository:

```
git clone https://github.com/ferreirajose/users-vr

```
------------------------

### Acessando diretorio do projeto

```
cd users-vr 
```
------------------------

### Install dependencies:

```
npm install
````

------------------------

### Executando o projeto

```
npm start
````
